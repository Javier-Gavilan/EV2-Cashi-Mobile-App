# Login React Native App

Vídeo Demostrativo: https://www.youtube.com/watch?v=0r9VyOx19ps

Aplicación de Login simple hecha con:

- React Native
- TypeScript
- Expo
- Node

Características:

- Login mediante credenciales predeterminadas
- Validación de Login
- Navegación por tabs
- Pantalla de Perfil mostrando el correo

Instrucciones:

- Dirigirse al directorio *cd (Directorio)*
- Ejecutar *yarn install*
- Ejecutar *yarn start*
- Comprobar funcionamiento desde Expo

Uso de IA:
